COMO USAR:

1. VISUALIZAR NO NAVEGADOR:
   $ uv run app examples/analise-laboratorial
   → Acesse http://localhost:5000

2. CONSULTAR VIA SQLite:
   $ sqlite3 examples/analise-laboratorial/data/sqlite/vibecforms.db
   > SELECT COUNT(*) FROM acreditadores;

3. FAZER BACKUP:
   $ cp data/sqlite/vibecforms.db data/sqlite/vibecforms.db.backup

4. RESETAR DADOS:
   $ rm data/sqlite/vibecforms.db
   $ python3 populate_demo_data.py
